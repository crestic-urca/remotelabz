Sécurité Firewalling Cisco ASA

Vous trouverez ci-dessous le schéma de la topologie du réseau. Vous devez configurer le firewall afin de filtrer les accès au serveur Web à configurer sous Linux, sur la VM_3. La configuration du firewall peut se faire :
  • soit via le CLI sur la fenêtre intitulée CLI ASA
  • soit via l'interface graphique sous windows

Configuration de l'accès à la configuration en mode graphique de l'ASA

  • Configurer le Windows sur l'IP 10.10.10.1/24
  • Connectez-vous en CLI sur l'ASA
  • Configurer l'accès en http à l'ASA par les commandes : 
    username admin password admin
http server enable
http 10.10.10.1 255.255.255.0 management
  • Connectez-vous maintenant depuis un navigateur Web à l'adresse https://10.10.10.254
