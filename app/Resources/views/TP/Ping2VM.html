Test de configuration réseau sous linux

  • Connectez-vous avec les identifiants indiqués sur le terminal
  • Lister les interfaces réseaux avec la commande 
    ip link show
  • Identifier la carte réseau ethX (la valeur de X dépend de votre machine)
  • Saisir une adresse IP sur chaque système
    ip addr add A.B.C.D/24 dev ethX
  • Activez au besoin votre interface réseau
    ip link set dev ethX up
  • Vérifiez l'adresse IP que vous avez affecté
    ip addr show
  • Testez votre connexion entre les 2 systèmes un 
    ping
