Test de configuration réseau sous linux

  • Connectez-vous avec les identifiants indiqués sur le terminal
  • Lister les interfaces réseaux avec la commande 
    ip link show
  • Identifier la carte réseau ethX (la valeur de X dépend de votre machine)
  • Tapez
    dhclient ethX
    afin de récupérer une IP via le serveur DHCP configuré sur le réseau
  • Trouvez l'adresse IP que le serveur DHCP vous a attribuée en tapant la commande 
    ip addr show
  • Testez votre connexion réseau en effectuant un 
    ping www.google.fr
